# Sistema Financeiro & RH

Sistema web para gestão de profissionais de saúde e administrativos.

## Funcionalidades

- Cadastro completo de profissionais (dados pessoais, profissionais, financeiros)
- Especialidades: Psicologia, Fonoaudiologia, Psicopedagogia, Psicomotricidade, Terapeuta Ocupacional, Assistente Terapêutico, Estagiário, Serviços Gerais, Recepcionista, Assistente Administrativo, Auxiliar Financeiro
- Controle de conselhos profissionais (CRP, CREFITO, CREFONO, CREF)
- Aniversariantes com lembrete automático 7 dias antes
- Relatórios gerais e por especialidade

## Rodar localmente

```bash
npm install
npm start
```

Acesse: http://localhost:3000

## Deploy no Railway

1. Suba este repositório no GitHub
2. Acesse railway.app e crie um novo projeto a partir do GitHub
3. O Railway detecta automaticamente o Node.js e faz o deploy
